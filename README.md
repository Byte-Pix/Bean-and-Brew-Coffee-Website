# Cool and fun features that you might find boring but I found cool

I found out that Google Maps has an embed feature that I could paste directly into the webite. Currently it's set to Apple's headquarters (cuz WWDC is next week). This interactable map just makes the website look more like a cafe site.
## Nav Bar changing colours on hover
I have made the navbar change colours when the mouse hovers over a link before, however relearning it when coming back to HTML was probably the coolest thing. I am a simple human being.

The <input placeholder="input" style="width: 40px"> and <button>button</button> tag was a new thing to me. Not very interesting by themselves but OMG STYLING IT IS ACTUALLY MORE FUN than any other part of the website (at least so far).
